# Participant Quick Start Guide

Welcome to the GitHub Copilot Bootcamp! This guide will help you get started quickly.

## Prerequisites

- GitHub account with Copilot access
- IDE with GitHub Copilot extension installed (VS Code, Visual Studio, JetBrains, Neovim, etc.)
- Basic programming knowledge
- Git installed on your machine

## Getting Started

1. **Fork this repository** to your own GitHub account
2. **Clone your fork** to your local machine:
   ```bash
   git clone https://github.com/YOUR-USERNAME/GHCP-bootcamp-template.git
   cd GHCP-bootcamp-template
   ```

3. **Enable GitHub Copilot** in your IDE
   - VS Code: Install the "GitHub Copilot" extension
   - Ensure you're signed in with your GitHub account

4. **Review the weekly materials**:
   - Week 1: Getting Started & Scaffold Projects
   - Week 2: Features, Documentation & Security
   - Week 3: DevOps & Testing
   - Week 4: Refactoring Legacy Code

## Weekly Structure

Each week follows this pattern:
- **Monday**: Introduction & Concepts
- **Tuesday-Thursday**: Hands-on Labs
- **Friday**: Reflection & Sharing

## Daily Reflections

Use GitHub Issues to document your daily learnings:
- Navigate to Issues > New Issue
- Select "Daily Reflection" template
- Share insights, challenges, and wins
